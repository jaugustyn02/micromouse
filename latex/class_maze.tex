\doxysubsection{Detailed Description}
Represents a 2D maze used in the micromouse simulation. 

Inherits from \doxylink{class_maze_reader}{Maze\+Reader} and implements maze generation, wall management, and cell state tracking. The maze consists of a grid of \doxylink{class_cell}{Cell} objects.
