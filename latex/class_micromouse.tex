\doxysubsection{Detailed Description}
Represents the \doxylink{class_micromouse}{Micromouse} robot. 

This class encapsulates the functionality of the \doxylink{class_micromouse}{Micromouse} robot, including its sensors, brain, and movement logic. It provides methods to control the mouse\textquotesingle{}s behavior and retrieve its current state.
