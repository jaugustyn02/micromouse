\doxysubsection{Detailed Description}
Concrete implementation of the \doxylink{class_simulation_controller}{Simulation\+Controller} interface. 

Manages the \doxylink{class_micromouse}{Micromouse} simulation, including maze generation, mouse movement, speed control, and step-\/by-\/step execution.
